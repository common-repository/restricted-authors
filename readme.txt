Easily restrict what Categories each author can publish in.

 == Description ==
Restricted Authors allows you to restrict which category each author can publish in, as well as set a default category on a per author basis.

Simply navigate to the edit users page and select the author whom you wish to restrict categories on. Select the categories and save the user settings.
